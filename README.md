# subwaive
subwaive is the internal membership tracker for the [MakeFixHack](https://makefixhack.org) makerspace. It links to Stripe and Docuseal to handle membership subscriptions and waivers, respectively.

## Setup and Development
This is a Django project written in Python running in Docker containers. To work on this project locally, follow the setup guide found in [setup.md](subwaive/setup.md).
